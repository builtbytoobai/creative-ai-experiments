# Project Brief

## Project name

creative-ai-experiments

## Purpose

Build practical AI tools and workflows for creative development, prompt engineering, music, storytelling, and AI-assisted software building.

This project is part of the builtbytoobai AI + creative tech portfolio.

## Target user

Developers and creatives exploring AI-assisted workflows, prompt engineering, and generative tools.

## Problem this solves

There is no single, well-structured place to experiment with and document AI-assisted creative and development workflows — this repo provides that foundation.

## Current status

- [ ] Idea
- [ ] Prototype
- [x] Working MVP
- [x] Public repo
- [ ] Product candidate

## Current repos in portfolio

- AI-Lyric-Generator
- AI-Prompt-Library
- creative-ai-experiments

## Tech stack

Markdown, Python, GitHub Actions, Claude Code, ChatGPT, Gemini

## Important decisions

- Use `.ai/` for live project AI instructions
- Use `ai-dev-workflow/` for reusable workflow templates
- Multi-agent review: Claude Code builds, ChatGPT reviews architecture, Gemini reviews product/UX

## Out of scope for now

- Production deployment
- User-facing UI or web app
- Monetization

## Success criteria

This project is successful when:

- AI workflow templates are reusable across new repos
- Claude Code can be onboarded to any new project using the templates
- The repo serves as a clear reference for AI-assisted development practices
